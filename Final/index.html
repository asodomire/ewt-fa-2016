Let's learn some Snap.svg

What is Snap.svg?

Snap.svg is a Javascript library that can manipulate/animate svgs.

It's basically like the jquery of working with SVGs.

What is an .svg?

"is an XML-based vector image format for two-dimensional graphics with support for interactivity and animation." -wikipedia 2k16

It's basically a bunch of code that you can use to put a vector on a web page, then use it for animations and stuff. -ashley 2k16
